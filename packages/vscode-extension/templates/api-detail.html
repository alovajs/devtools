GET
/api/v1/users/{userId}/orders
获取指定用户的所有订单信息，包括订单状态、商品列表、价格等详细信息。支持分页和过滤条件。
Parameters
Response
Path Parameters
参数名 必填 类型 描述
userId 必填 integer 用户唯一标识ID，必须是有效的用户ID
Query Parameters
参数名 必填 类型 描述
status 可选 string 订单状态过滤条件，可选值: pending, completed, cancelled
page 可选 integer 分页页码，默认值为1
limit 可选 integer 每页记录数，默认值为20，最大值为100
sort 可选 string 排序字段，格式: field_name:asc 或 field_name:desc
Request Body
application/json
此接口不需要请求体
200: Success 400: Bad Request 401: Unauthorized 404: Not Found
