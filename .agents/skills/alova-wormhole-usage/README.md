# Alova OpenAPI Integration Skill

Guide for integrating [alova](https://alova.js.org/) with OpenAPI/Swagger specs via `@alova/wormhole` to auto-generate API request functions and TypeScript types.

> For client-side request usage, see `alova-client-usage` skill.

## Supported Platforms

Vue, React, Svelte, Solid, Uniapp, Taro, and any framework supported by alova

## Trigger Scenarios

Invoke this skill when working with:

- OpenAPI/Swagger spec integration with alova
- `@alova/wormhole` configuration and usage
- API code generation from OpenAPI specs
- Wormhole plugins (rename, tagModifier, payloadModifier, filterApi, etc.)
- alova devtools or VSCode extension configuration
- `alova gen` / `alova init` CLI commands

## Documentation Structure

- **SKILL.md** — Configuration guide, plugin reference, and handleApi hook usage
- **references/INSTALLATION.md** — Installation and setup guide
- **references/CLI.md** — CLI command reference
- **references/PROGRAMMATIC-API.md** — Programmatic API usage
- **references/ALOVA-INSTANCE.md** — Customizing the alova instance

## Usage

For specific configuration or plugin questions, the skill fetches official docs on demand to ensure accuracy.
