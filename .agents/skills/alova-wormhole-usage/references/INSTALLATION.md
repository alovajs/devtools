# Installation & Setup

## Install packages

```bash
# npm
npm install @alova/wormhole --save-dev

# yarn
yarn add @alova/wormhole --dev

# pnpm
pnpm add @alova/wormhole -D
```

## VSCode extension (optional)

Search **"alova"** in the VSCode Extensions marketplace and install it. Provides:

- In-editor API documentation
- Quick navigation to interfaces
- Visual parameter and response structure preview

> ⚠️ Only VSCode liked editors are supported. For WebStorm and other editors, use the CLI (`npx alova gen`) instead.
