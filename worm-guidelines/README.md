# worma — Universal OpenAPI Code Generator

Skill for configuring and using [worma](https://worma.js.org) to auto-generate API calling functions, TypeScript types, in-editor documentation, and AI Skills from OpenAPI/Swagger specs.

## Trigger Scenarios

Invoke this skill when working with:

- worma configuration (`worma.config.js`, `.wormarc`)
- API code generation from OpenAPI specs
- `npx worma init` / `npx worma gen` CLI commands
- worma plugins (platform, apifox, rename, tagModifier, aiDoc, etc.)
- Template selection (alova, axios, fetch, ky, alovaGlobals, custom)
- Migration from `@alova/wormhole` to worma
- VSCode extension for in-editor API documentation
- AI Skills generation for coding agents
