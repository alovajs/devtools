# Customizing the Request Instance

→ Full reference: [Predefined Templates](https://worma.js.org/docs/template-system/predefined-templates.md)

The entry file generated by worma is preserved across regenerations — customize your request instance there.

## Alova Template

```ts
import { getUserList } from './api/services/user';
const users = await getUserList({ params: { page: 1 } });
```

## AlovaGlobals Template

```ts
// Mount once in entry file
import './api/alova-globals';
// Use globally
const user = await Apis.user.getUserById({ pathParams: { id: 1 } });
```

## Axios Template

```ts
import { createUser } from './api/axios/services/user';
const newUser = await createUser({ data: { name: 'Alice' } });
```

## Fetch / Ky Templates

```ts
import { getUserList } from './api/fetch/services/user';
const users = await getUserList({ params: { page: 1 } });
```
