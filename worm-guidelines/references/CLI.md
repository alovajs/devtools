# CLI Reference

→ Full reference: [CLI Commands](https://worma.js.org/docs/cli-commands.md)

## worma gen

```bash
npx worma gen                  # Workspace mode
npx worma gen -f               # Force regenerate
npx worma gen -p ./packages/app # Specific project
```

## worma init

```bash
npx worma init
npx worma init -t typescript
npx worma init -T axios
npx worma init -p ./packages/app
```
