# worma 官网首页设计方案 V2.6

worma 是一款基于 OpenAPI 的代码生成工具，为开发者输出类型安全的接口代码，为 AI 输出结构化接口知识。

---

## 一、内容设计（8 板块）

### 板块 1：Hero 主视觉区

- **主标题**：「一份 OpenAPI，从人类到AI」
- **副标题**：「为你生成类型安全的接口代码，为AI生成易理解的接口知识」
- **CTA 按钮组**：
  - 主按钮：「即刻体验」（跳转 `/docs/quick-start`）
  - 次按钮：「快速开始」（跳转 `/docs/quick-start`）

### 板块 2：核心特性区（2×2 网格）

- **标题**：「为什么选择 worma」
- 四张特性卡片：
  - **4 种产物，一次生成** —— 「接口代码、类型定义、接口文档、AI Skill，一条命令全部输出」
  - **通用化设计** —— 「4 种预设模板开箱即用，自定义模板灵活扩展」
  - **灵活可控** —— 「生成前编辑 OpenAPI 数据，按需过滤与调整」
  - **编辑器内文档** —— 「JSDoc 注释悬停查阅，JavaScript 项目同样获得类型提示」

### 板块 3：三步流程区

- **标题**：「从 OpenAPI 到生产代码」
- **三步流程**（纵向递进，每步配图标与简述）：
  - 步骤 1：「交付 OpenAPI 文件」
  - 步骤 2：worma 图标（视觉锚点，处理引擎）
  - 步骤 3：「一条命令，4 种产物」—— worma 将 OpenAPI 转化为：
    - **接口代码** —— 类型安全的调用函数
    - **类型定义** —— 完整的 TypeScript 类型声明
    - **接口文档** —— 结构化的 API 参考
    - **AI Skill** —— AI 可读的接口知识

### 板块 4：插件系统

- **标题**：「9 款内置插件，按需组合」
- **副标题**：「插件按顺序执行，灵活编排生成流程」
- **核心插件**（4 项，横向卡片展示）：
  - **rename** —— 批量重命名 API 函数与参数，支持多种命名风格
  - **filterApi** —— 按 URL 或 tag 过滤接口，只生成需要的部分
  - **importType** —— 复用项目已有的类型定义，避免重复生成
  - **aiDoc** —— 自动生成 AI Skill 文档，让 AI 读懂你的 API

### 板块 5：适配矩阵区

- **标题**：「跨栈适配」
- 三列横向布局：
  - 左列：「后端语言」—— Java、Go、Python、Node.js、C#
  - 中间：「worma 图形」（视觉锚点）
  - 右列：「前端请求库」—— Alova、Axios、Fetch、Ky
- **配文**：「支持主流技术栈」

### 板块 6：编辑器扩展区

- **标题**：「编辑器内，接口信息即时可得」
- **功能亮点**（左列竖排）：
  - **侧边栏 API 浏览器** —— 树形结构浏览所有接口（配图先占位）
  - **悬停文档** —— 鼠标悬停即显示接口详情（配图先占位）
  - **传送门** —— 调用代码上方按钮直达接口文档
  - **快速插入调用** —— 输入 URL 或描述，自动匹配并插入调用函数
  - **自动检测** —— 监听 OpenAPI 文件变更，实时更新（配图先占位）
  - **JS 类型提示** —— JavaScript 项目的智能补全
