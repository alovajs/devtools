# worma 官网首页设计方案

worma是一款基于OpenAPI的代码生成工具，可以自动生成TypeScript代码、接口文档、AI Skill等。

---

## 一、内容设计

### 板块 1：Hero 主视觉区

- **主标题**：「一份 OpenAPI，两个世界」 / _One Spec, Two Worlds_
- **副标题**：「worma —— 正如虫洞连接宇宙的两端，我们将后端接口与前端代码，开发者与 AI，无缝桥接」
- **CTA 按钮组**：
  - 主按钮：「即刻体验」（跳转 StackBlitz 示例）
  - 次按钮：「快速开始」（跳转 `/docs/quick-start`）
  - 辅助入口：「npx skills add alovajs/skills --skill worma」
- **核心价值标语**：滚动浮现「生成调用代码 · TypeScript 类型 · 接口文档 · AI Skill」

### 板块 2：核心问题陈述区

- **标题**：「AI 写代码，接口总出错？」
- **痛点陈列**（三幅微型"画框"）：
  - 画框 1：函数名猜错 → 运行时报错
  - 画框 2：参数不匹配 → 反复试错
  - 画框 3：AI 不了解你的 API → 凭空捏造
- **转折语**：「而这，正是 worma 诞生于世的理由」

### 板块 3：工作原理区

- **标题**：「虫洞的法则」
- **三步流程图**（对应 mdx 中的 `<Steps>`）：
  - 步骤 1：「后端交付 OpenAPI 文件」—— Swagger / Knife4j / FastAPI 等
  - 步骤 2：「`npx worma gen` 一键生成」—— `.skills/` + `src/api/` 双产出
  - 步骤 3：「AI 自动补全，精准调用」—— 描述意图即得代码
- 每一步配有生成的代码预览片段（`.skills/SKILL.md` 和 `src/api/user.ts` 的视觉效果）

### 板块 4：核心特性展示区

- **标题**：「四把密钥，开启虫洞」
- 四张"特性卡片"（对应四合一输出 + 三大特性）：
  - 🤖 **AI Skill 一键生成** —— 「AI 读懂你的所有 API」
  - 🧩 **四合一信息输出** —— 「一份文档，四方受益」
  - 🔌 **多请求库 + 自定义模板** —— 「Alova / Axios / Fetch / Ky，自由切换」
  - 📝 **编辑器内文档** —— 「悬停即阅，JavaScript 也能享受类型提示」

### 板块 5：适配矩阵区

- **标题**：「万物皆可连通」
- 以左右两列展示：
  - 左侧：「后端语言」—— Java、Go、Python、Node.js ……
  - 中间：「worma 虫洞」（视觉锚点）
  - 右侧：「前端请求库」—— Alova、Axios、Fetch、Ky ……
- 配文：「无论你站在哪一端，worma 都能打通你与 AI 之间的最后一道壁垒」

### 板块 6：即刻体验区

- **标题**：「穿越虫洞，即刻体验」
- 四个在线示例入口卡片：
  - TypeScript 示例
  - ESM 示例
  - CommonJS 示例
  - Monorepo 示例
- 每张卡片为 StackBlitz 一键跳转

### 板块 7：底部召唤区

- **标题**：「为你的 AI 装上 worma 之眼」
- 命令行展示：
  ```
  npx skills add alovajs/skills --skill worma
  ```
- **结语**：「安装完成，AI 便能理解 worma 的配置与用法。—— 虫洞已开启」
- 底部导航链接：GitHub、文档、社区

## 二、排版设计

整体采用 **文艺复兴绘画的经典构图法则**，以对称、金字塔式、黄金分割比例为核心骨架。

### 整体布局结构

```
┌─────────────────────────────────────────────────┐
│                [Frame - 顶部金色边框]              │
│  ┌─────────────────────────────────────────────┐│
│  │        HERO（金字塔式构图，视觉重心居中）       ││
│  │       主标题 → 副标题 → CTA → 浮现金句        ││
│  └─────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────┐│
│  │     问题陈述区（三联画 Triptych 布局）         ││
│  │   [画框1]  [画框2]  [画框3]  →  转折语       ││
│  └─────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────┐│
│  │    工作原理区（纵向三幕式，古典叙事节奏）       ││
│  │       步骤1 → 步骤2 → 步骤3                  ││
│  │         (螺旋递进，由浅入深)                   ││
│  └─────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────┐│
│  │    核心特性区（对称四宫格）                    ││
│  │    [卡片A]          [卡片B]                  ││
│  │    [卡片C]          [卡片D]                  ││
│  └─────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────┐│
│  │    适配矩阵（三联拱门布局）                    ││
│  │   [后端]  ──  [worma]  ──  [前端]           ││
│  └─────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────┐│

│  └─────────────────────────────────────────────┘│
│  ┌─────────────────────────────────────────────┐│
│  │      底部召唤区（正中穹顶式收束）              ││
│  │         命令行 → 结语 → 导航                  ││
│  └─────────────────────────────────────────────┘│
│                [Frame - 底部金色边框]              │
└─────────────────────────────────────────────────┘
```

### 排版细节

| 元素         | 排版规则                                        |
| ------------ | ----------------------------------------------- |
| **主标题**   | 居中，大号衬线体（Serif），字距宽松，仿古典铭文 |
| **副标题**   | 居中，小字号，斜体（Italic），人文主义风格      |
| **段落标题** | 居中，带左右装饰线（类似古典卷轴标题）          |
| **正文**     | 两侧留白充裕，行高 1.8-2.0，仿手抄本排版呼吸感  |
| **卡片**     | 四角带弧形装饰，内文左对齐，图标居中偏上        |
| **间距**     | 板块间距极大（120px+），模拟绘画的"留白即呼吸"  |
| **视差**     | Hero 区与下方内容区之间使用缓动视差，营造景深感 |

## 三、风格设计

### 核心视觉关键词

**「油彩 · 金箔 · 穹光 · 虫洞」**

### 色彩体系

| 角色                   | 色值                       | 用途                                 |
| ---------------------- | -------------------------- | ------------------------------------ |
| **主色（天青石蓝）**   | `#0050ff`                  | 主按钮、标题强调、虫洞视觉、链接悬停 |
| **辅色（威尼斯红）**   | `#8B2500`                  | 次要强调、装饰线、热点标记           |
| **金色（金箔）**       | `#D4A843` → `#F2D06B` 渐变 | 边框装饰、分隔线、图标描边、画框     |
| **深底色（古典画布）** | `#1A1520`                  | 页面背景，模拟古典油画的深色底层     |
| **暖白（羊皮纸）**     | `#F5F0E8`                  | 正文文字色，模拟羊皮纸质感           |
| **暗金（古铜）**       | `#8B7536`                  | 次级边框、卡片底色                   |
| **光晕白**             | `#FFFEF5`（带透明度）      | 高光、文字高亮、虫洞光效             |

### 质感体系

#### 1. 画布纹理

- 整体背景叠加细微的 **油画布纹理**（canvas texture），模拟亚麻画布的经纬肌理
- 深色区域叠加 **伦勃朗式暗调**，边缘自然渐暗（vignette 效果）

#### 2. 金箔边框

- 每个内容板块外围使用 **金色装饰框**，模拟古典祭坛画（altarpiece）的镀金框架
- 边框四角带有 **蔓草纹（arabesque）装饰角花**
- 分割线采用金色双线 + 中心菱形节点

#### 3. 光照系统（Chiaroscuro）

- Hero 区域使用 **穹顶光**（上方投射的聚光效果），模拟教堂穹顶天窗
- 标题文字带有 **金色内发光**（inner glow），模拟金箔在侧光下的反光
- 卡片元素带有 **柔和的底部阴影**，模拟油画中物体的厚度感

#### 4. 虫洞视觉元素

- 核心图形为 **螺旋涡旋**（spiral vortex），以 `#0050ff` 为主色，向中心渐变为亮白
- 涡旋周围散布 **金色星点**，模拟虫洞能量
- 在 Hero 区作为背景水印，适配矩阵区作为中心连接符号

#### 5. 字体体系

| 用途           | 字体                                          | 风格                     |
| -------------- | --------------------------------------------- | ------------------------ |
| 主标题         | **Playfair Display** / **Cormorant Garamond** | 经典衬线体，文艺复兴气质 |
| 副标题/引言    | **Italic 变体**                               | 人文主义意大利斜体       |
| 正文           | **Crimson Text** / **EB Garamond**            | 高可读性古典衬线体       |
| 代码/命令行    | **JetBrains Mono** / **Fira Code**            | 等宽字体，与古典形成对比 |
| 中文（如需要） | **思源宋体** / **Noto Serif SC**              | 中文衬线体，匹配古典氛围 |

#### 6. 动效体系

| 动效                  | 描述                                               |
| --------------------- | -------------------------------------------------- |
| **油彩晕开（Bloom）** | 元素入场时带有柔和的扩散光晕，模拟颜料在画布上晕染 |
| **金箔闪烁**          | 边框/装饰元素带有微弱的金色光斑流动效果            |
| **虫洞旋转**          | 中心涡旋持续缓慢旋转，星点环绕运动                 |
| **视差景深**          | 滚动时前景与背景以不同速率移动，模拟绘画空间层次   |
| **笔触浮现**          | 文字入场时模拟画笔一笔一笔写出的效果               |

### 风格参考

- **卡拉瓦乔**：戏剧性明暗对照（Chiaroscuro）
- **扬·凡·艾克**：精细的金色装饰与珐琅质感
- **拉斐尔**：和谐对称的构图
- **维米尔**：柔和的天光投射

## 四、视觉风格总结

> worma 首页将如同 **一幅被镶在金框中的文艺复兴三联祭坛画**——
>
> 深色的画布上，天青石蓝的虫洞涡旋在穹顶之光中缓缓旋转，金色的蔓草纹缠绕着每一个内容板块的边框。开发者在左，AI 在右，worma 居于正中，如同圣像画中连接天与地的桥梁。代码以羊皮纸般的暖白色呈现，命令行则是镶嵌在金色凹槽中的碑文。
>
> 这不仅是官网，更是一幅数字时代的 **「技术祭坛画」**——庄重而鲜活，古典而锐利。
