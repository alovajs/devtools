## defineConfig

```typescript
function defineConfig(config: Config): Config;
```

## Config

```typescript
interface Config {
  /** 生成规则数组 */
  generator: GeneratorConfig[];
}
```

## GeneratorConfig

```typescript
interface GeneratorConfig {
  /** OpenAPI 文档地址（URL 或本地路径），支持 string[] 依次尝试 */
  input: string | string[];

  /** 输出目录 */
  output?: string;

  /** 是否生成文档注释，默认 true */
  docComment?: boolean;

  /** 响应 MediaType，可设置数组，表示依次指向的 media type */
  responseMediaType?: string | string[];

  /** 请求体 MediaType，可设置数组，表示依次指向的 media type */
  bodyMediaType?: string | string[];

  /** 多文档时的自定义服务名称，未设置时默认为 info.title */
  serverName?: string;

  /** 插件数组（模板也通过 plugin 的 getTemplate hook 设置） */
  plugins?: ApiPlugin[];

  /** 性能配置 */
  performance?: PerformanceConfig;
}
```

## PerformanceConfig

```typescript
interface PerformanceConfig {
  /** schema→TS worker 池策略：'auto' | number | false。默认 'auto'（按接口数自适应） */
  workerPool?: 'auto' | number | false;
  /** transform 阶段并发上限。默认 auto（min(64, max(8, cpus*4))） */
  transformConcurrency?: number;
  /** 写盘并发数。默认 32 */
  writeConcurrency?: number;
  /** 是否对最终文件做 prettier 格式化。默认 true */
  formatFile?: boolean;
  /** 是否对输出做确定性排序（tag/API/component 按名称字典序）。默认 true */
  deterministicSort?: boolean;
}
```

## GenerateApiOptions

```typescript
interface GenerateProgress {
  /** 'core' 表示框架核心；其他值为插件 name */
  source: string;
  /** 0-100 百分比 */
  progress: number;
  /** 可选消息 */
  message?: string;
}

interface GenerateApiOptions {
  /** 强制重新生成，忽略缓存 */
  force?: boolean;
  /** 项目路径 */
  projectPath?: string;
  /** 进度回调。框架按 progressInterval 节流后推送所有 source 的最新进度 */
  onProgress?: (snapshot: Record<string, GenerateProgress>) => void;
  /** 推送间隔（毫秒），默认 500。<= 0 时关闭节流 */
  progressInterval?: number;
}
```

## 环境变量覆盖

| 环境变量 | 说明 |
|---------|------|
| `WORMA_WORKERS=auto\|N\|0` | 覆盖 `performance.workerPool` |
| `WORMA_PRETTIER=0\|1` | 覆盖 `performance.formatFile` |

## 配置变化（v1 → v2）

以下配置项已在 v2 中移除：

| 移除的配置项 | 替代方案 |
|------------|---------|
| `template` | 改为插件方式，使用 `plugins` 数组中的模板插件 |
| `handleApi` | 使用 `openapiParsed` 钩子替代 |
| `fileNameCase` | 已移除 |
| `autoUpdate`（根配置） | VSCode 扩展自行管理 |
| `platform` 参数 | 改为 `platform` 插件 |
