worma 是一款完全开源、MIT 协议发布的 OpenAPI 代码生成工具。所有功能永久免费，由社区贡献者用业余时间维护。

## 为什么需要你的支持

开源项目看似「免费」，但持续的开发和维护需要大量时间与精力：

- **功能迭代**：适配新的前端框架、请求库和后端语言生态
- **文档维护**：多语言文档编写、示例代码更新、视频教程制作
- **社区支持**：GitHub Issues / Discussions 回复、PR Review
- **基础设施**：域名、CDN、CI/CD 服务器成本

你的每一份支持，都能让维护者更专注地投入开发，让 worma 走得更远。

## 赞助权益

### 🌟 个人赞助（$5 / 月）

- 在 GitHub 上获得 `Sponsor` 徽章
- 优先处理你提交的 Issue
- 加入专属 Discord 频道

### 💎 金牌赞助（$50 / 月）

- 包含个人赞助全部权益
- 你的名字 / Logo 将展示在 README 和文档首页
- 每季度一次 30 分钟 1v1 技术咨询

### 🏢 企业赞助（$200 / 月）

- 包含金牌赞助全部权益
- 官网首页 Logo 展示（带链接）
- 优先功能需求评估与排期
- 每年一次 2 小时团队技术培训
- 开具可用于抵扣税款的发票

> 所有赞助均为自愿行为，不影响 worma 的任何功能使用。赞助费用将 100% 用于项目开发与维护。

## 其他支持方式

如果不方便直接赞助，以下方式同样重要：

- **代码贡献**：[GitHub Issues](https://github.com/alovajs/devtools/issues) 中查找 `good first issue` 标签
- **文档翻译**：帮助完善多语言文档
- **分享推广**：在技术社区、博客、社交媒体分享 worma
- **Bug 反馈**：提交高质量的 Bug Report 或 Feature Request
- **Star**：给 [alovajs/devtools](https://github.com/alovajs/devtools) 点个 Star

每一项贡献，无论大小，都值得被铭记。
