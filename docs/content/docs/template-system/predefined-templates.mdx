## alova（functional）

按需引入模式，每个 API 作为独立函数导出，支持 Tree-shaking。

```javascript
import { alova } from 'worma/plugin';

plugins: [alova({ framework: 'vue' })]
```

**模板结构：**

```text
alova-functional/typescript/
├── index.ts.handlebars       # alovaInstance + 重导出
├── {tag}.ts.handlebars       # 按 tag 生成 API 函数
└── types.ts.handlebars       # 类型声明
```

**调用示例：**

```typescript
import { getUser } from './api/user';
const user = await getUser({ params: { id: 1 } });
```

## alovaGlobals

全局挂载模式，兼容 alova@3（仅 v3，不再支持 v2）。

```javascript
import { alovaGlobals } from 'worma/plugin';

plugins: [alovaGlobals({
  global: 'Apis',
  globalHost: 'globalThis',
})]
```

**参数说明：**

```typescript
interface TemplateAlovaGlobalsConfig {
  /** 全局导出的 api 名称，默认为 'Apis'，多 generator 时为必填且不可重复 */
  global?: string;
  /** 全局 api 对象挂载的宿主对象，默认为 'globalThis' */
  globalHost?: string;
  /** 使用 import type 方式导入，默认 false */
  useImportType?: boolean;
}
```

> `global` 参数会拼接在每个 API 的 `pathKey` 和 `defaultValue` 最前面（如 `Apis.pet.addPet`），确保全局引用路径正确。

**调用示例：**

```typescript
const users = await Apis.user.getUserList({ params: { page: 1 } });
```

## axios

生成基于 axios 的调用函数，采用按需引入模式。

```javascript
import { axios } from 'worma/plugin';
plugins: [axios()];
```

**模板结构：**

```text
axios/typescript/
├── index.ts.handlebars       # axiosInstance + 重导出
├── {tag}.ts.handlebars       # 按 tag 生成 API 函数
└── types.ts.handlebars       # 类型声明
```

**调用示例：**

```typescript
import { getUser } from './api/user';
const user = await getUser({ params: { id: 1 } });
```

## fetch

```javascript
import { fetch } from 'worma/plugin';
```

**模板结构：**

```text
fetch/typescript/
├── index.ts.handlebars       # BASE_URL + 重导出
├── {tag}.ts.handlebars       # 按 tag 生成 API 函数
└── types.ts.handlebars       # 类型声明
```

**调用示例：**

```typescript
import { getUser } from './api/user';
const user = await getUser({ params: { id: 1 } });
```
